## 项目背景与意义

### 1.1 研究背景

在数字化时代，大量纸质文档需要转换为电子格式进行存储和处理。然而，由于扫描设备质量、文档保存状况、拍摄条件等因素的影响，获取的文档图像往往存在以下问题：

1. **分辨率低** - 文字模糊不清，难以识别
2. **文字缺失** - 文档老化、污损导致文字丢失
3. **识别错误** - OCR 识别存在错别字
4. **质量参差不齐** - 需要人工逐份检查

传统的人工处理方式效率低下，成本高，无法满足大规模文档数字化的需求。因此，开发一套智能化的文档修复与增强系统具有重要的现实意义。

### 1.2 研究意义

**理论意义**:
- 探索多模型协同的文档处理方法
- 研究基于深度学习的文本纠错机制
- 构建完整的文档质量评估体系

**实际意义**:
- 提高文档数字化效率
- 降低人工成本
- 提升文档质量
- 保护历史文献资料

#### 4.3.2 MacBERT 严格模式

**核心思想**: 只纠正很确定的错误，保护原文格式

**实现步骤**:

```python
def _macbert_correct(self, corrector_dict, text: str):
    # 1. 统计中文字符数量
    chinese_chars = re.findall(r'[\u4e00-\u9fa5]', text)
    if len(chinese_chars) < 5:
        return text, []  # 中文字符太少，跳过
    
    # 2. 提取非中文字符，记录位置
    non_chinese_segments = []
    for match in re.finditer(r'[^\u4e00-\u9fa5]+', text):
        non_chinese_segments.append({
            'start': match.start(),
            'end': match.end(),
            'text': match.group()
        })
    
    # 3. 提取纯中文文本
    pure_chinese = re.sub(r'[^\u4e00-\u9fa5]+', '', text)
    
    # 4. 应用的/地/得规则
    de_corrections = self._apply_de_rule(pure_chinese)
    
    # 5. MacBERT 预测
    inputs = tokenizer(pure_chinese, return_tensors='pt')
    outputs = model(**inputs)
    predictions = torch.argmax(outputs.logits, dim=-1)
    
    # 6. 逐字比较，只纠正确定的错误
    corrected_chars = []
    for orig, pred in zip(original_ids, predicted_ids):
        if orig == pred:
            # 检查是否是的/地/得规则纠正
            if position in de_corrections:
                corrected_chars.append(de_corrections[position])
            else:
                corrected_chars.append(orig)
        elif self._is_common_confusion(orig, pred):
            corrected_chars.append(pred)  # 常见错误，纠正
        else:
            corrected_chars.append(orig)  # 不确定，保持原文
    
    # 7. 还原非中文字符
    result = self._restore_non_chinese(corrected_chars, non_chinese_segments)
    
    return result, errors
```

#### 4.3.3 的/地/得语法规则

**规则 1**: 动词 + 得 + 补语
```
跑的很快 → 跑得很快  # 动词 + 得 + 程度副词
打的很好 → 打得很好
写的很快 → 写得很快
```

**规则 2**: 形容词 + 地 + 动词
```
高兴的说 → 高兴地说  # 形容词 + 地 + 动词
慢慢的走 → 慢慢地走
认真的学 → 认真地学
```

**实现代码**:
```python
def _apply_de_rule(self, text: str) -> dict:
    corrections = {}
    
    # 规则 1: 动词 + 的 + 很/非常/特别 → 得
    pattern1 = r'(.+?) 的 (很 | 非常 | 特别 | 十分)'
    for match in re.finditer(pattern1, text):
        pos = match.start() + len(match.group(1)) + 1
        corrections[pos] = '得'
    
    # 规则 2: 形容词 + 的 + 动词 → 地
    adj_words = ['慢慢', '轻轻', '悄悄', '高兴', '认真', '仔细']
    pattern2 = r'([\u4e00-\u9fa5]{2,}) 的 ([\u4e00-\u9fa5])'
    for match in re.finditer(pattern2, text):
        adj = match.group(1)
        if any(adj.endswith(w) for w in adj_words):
            pos = match.start() + len(adj) + 1
            corrections[pos] = '地'
    
    return corrections
```

### 4.4 效果评估模块

#### 4.4.1 图像质量指标

**PSNR (峰值信噪比)**
```python
def calculate_psnr(original, enhanced):
    mse = np.mean((original - enhanced) ** 2)
    if mse == 0:
        return float('inf')
    max_pixel = 255.0
    psnr = 20 * log10(max_pixel / sqrt(mse))
    return psnr
```

**评价标准**:
- > 30dB: 优秀
- > 25dB: 良好
- > 20dB: 可用
- < 20dB: 较差

**SSIM (结构相似性)**
```python
def calculate_ssim(original, enhanced):
    # 亮度、对比度、结构比较
    ssim_value = ssim(original, enhanced, multichannel=True)
    return ssim_value
```

**评价标准**:
- > 0.9: 优秀
- > 0.8: 良好
- > 0.7: 可用
- < 0.7: 较差

#### 4.4.2 文本质量指标

**CER (字符错误率)**
```python
def calculate_cer(reference, hypothesis):
    """
    CER = (S + D + I) / N
    S: 替换次数
    D: 删除次数
    I: 插入次数
    N: 总字符数
    """
    distance = edit_distance(reference, hypothesis)
    cer = distance / len(reference)
    return cer
```

**评价标准**:
- < 5%: 优秀
- < 10%: 良好

# backend/app/core/security.py
def create_access_token(data: dict, expires_delta: timedelta = None):
    to_encode = data.copy()
    if expires_delta:
        expire = datetime.utcnow() + expires_delta
    else:
        expire = datetime.utcnow() + timedelta(minutes=15)
    
    to_encode.update({"exp": expire})
    encoded_jwt = jwt.encode(to_encode, SECRET_KEY, algorithm=ALGORITHM)
    return encoded_jwt

def verify_token(token: str):
    try:
        payload = jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])
        return payload
    except jwt.ExpiredSignatureError:
        return None
    except jwt.InvalidTokenError:
        return None
```

# backend/app/routes/batch.py
@router.post("/create")
async def create_batch(
    files: List[UploadFile],
    task_type: str,
    current_user: User = Depends(get_current_user)
):
    # 1. 创建批量任务
    batch = BatchTask(
        user_id=current_user.id,
        total_files=len(files),
        status='queued'
    )
    db.add(batch)
    db.commit()
    
    # 2. 后台异步处理
    background_tasks.add_task(
        process_batch,
        batch_id=batch.id,
        files=files,
        task_type=task_type
    )
    
    return {"batch_id": batch.id, "status": "queued"}

# 后台处理函数
def process_batch(batch_id: int, files: List[UploadFile], task_type: str):
    for i, file in enumerate(files):
        try:
            # 处理单个文件
            result = process_single_file(file, task_type)
            
            # 更新进度
            update_batch_progress(batch_id, i + 1)
        except Exception as e:
            log_error(batch_id, e)
    
    # 完成
    complete_batch(batch_id)
```
